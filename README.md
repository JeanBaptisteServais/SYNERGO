And distance beetween finger's are egal to

    D(Fi, Fi+1) = (w * 0.295) * x
    
    where i ∈ N and i ∈ [1; 4] and x ∈ N and i ∈ [1; 4]
    
 

    or h * ... in course

    - where w, h are length and hight of hand position,
